This application will be used to import publications and allow users to subscribe.

## Setup

1. Clone this repository
1. `bundle`
1. Create databases: `rake db:create`
1. Run migrations: `rake db:migrate`
1. Run Specs `rspec`
1. Run Local Server `rails s`
